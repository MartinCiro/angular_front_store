Algoritmos de Clasificación: Una Comparativa

Por Jane Doe | 21 de Mayo, 2024

Fundamentos Teóricos

Los algoritmos de clasificación son un componente fundamental en la informática y el análisis de datos. Su objetivo principal es organizar un conjunto de elementos en un orden específico, ya sea numérico, alfabético o basado en algún otro criterio. La eficiencia de un algoritmo de clasificación se mide comúnmente por su complejidad temporal y espacial. En este estudio, nos centraremos en dos algoritmos populares: Bubble Sort y Quick Sort, para analizar sus diferencias en rendimiento y aplicación.

Ejemplos de Código

A continuación, se presentan las implementaciones en Python para ambos algoritmos. El código está diseñado para ser claro y demostrativo, facilitando la comprensión de su lógica interna.

Bubble Sort


def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

Quick Sort


def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[len(arr) // 2]
        left = [x for x in arr if x < pivot]
        middle = [x for x in arr if x == pivot]
        right = [x for x in arr if x > pivot]
        return quick_sort(left) + middle + quick_sort(right)
